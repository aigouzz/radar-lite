# 📡 TrendRadar Lite - 本地抓取版

全网热搜聚合看板，**前后端一体**，本地运行，无需任何第三方 API Key。

## 架构

```
┌─────────────┐      /api/weibo       ┌─────────────────┐
│  React 前端  │  ──────────────────▶  │  Express 后端    │  ──▶  微博/百度/知乎 ...
│  :3000       │  ◀──────────────────  │  :3001           │  ◀──  各平台公开接口
│  (Vite)      │      JSON 数据        │  (Node.js)       │
└─────────────┘                        └─────────────────┘
```

- **后端** (`server/index.js`): Express 服务，直接请求各平台公开接口抓取热搜
- **前端** (`src/App.jsx`): React 页面，通过 Vite 代理请求后端
- **`npm run dev`** 一条命令同时启动前后端

## 快速启动

```bash
# 1. 安装依赖
npm install

# 2. 启动（前后端同时启动）
npm run dev
```

浏览器自动打开 `http://localhost:3000`

## 常见问题

**Q: 某些平台显示"抓取失败"？**
A: 部分平台有反爬策略，在国内网络环境下大部分平台可正常访问。

**Q: 如何添加新平台？**
A: 在 `server/index.js` 中添加抓取函数并注册到 `crawlers` 对象。

**Q: 如何修改缓存时间？**
A: 修改 `server/index.js` 中的 `CACHE_TTL`（默认 60 秒）。
